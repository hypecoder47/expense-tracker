# Expense Tracker

A command-line expense tracker built with Python.

## What it does
- Add daily expenses with name and amount
- Saves data to a JSON file
- View total spending summary

## How to run
1. Make sure Python is installed
2. Clone this repo
3. Run: python expense_tracker.py

## Built with
- Python 3
- JSON for data storage
